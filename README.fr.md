# PDFTranslator

**PDFTranslator** est un traducteur PDF gratuit basé sur l'IA, conçu pour traduire des documents PDF en ligne tout en conservant autant que possible la mise en page d'origine.

Site officiel : [https://pdftranslator.org/](https://pdftranslator.org/)

[English](README.md) | [中文](README.zh-CN.md) | [Deutsch](README.de.md) | Français | [日本語](README.ja.md) | [한국어](README.ko.md)

## À propos de PDFTranslator

PDFTranslator.org aide les utilisateurs à traduire des fichiers PDF directement dans le navigateur, sans installer de logiciel de bureau. Il est conçu pour les documents dont la mise en page est importante, notamment les articles de recherche, les manuels scolaires, les guides, les contrats, les rapports, les modes d'emploi et les documents professionnels.

Le service se concentre sur la traduction PDF par IA, la traduction PDF avec OCR et la traduction de documents avec conservation du format dans plus de 100 langues.

## Fonctionnalités clés

- **Traduction PDF gratuite en ligne** : Traduisez des documents PDF sans installer de logiciel.
- **Aucune inscription requise** : Commencez à traduire sans créer de compte.
- **Conservation du format** : Conserve autant que possible la mise en page, les tableaux, les images, les polices et la structure du PDF d'origine.
- **Aperçu bilingue comparatif** : Vérifiez le texte original et le texte traduit côte à côte avant de télécharger le résultat final.
- **Options de style de traduction** : Choisissez un style de traduction adapté au type de document, par exemple académique, professionnel, formel ou naturel.
- **Traduction PDF avec OCR** : Traduisez des documents PDF scannés ou basés sur des images grâce à la prise en charge de l'OCR (bientôt disponible).
- **Plus de 100 langues** : Traduisez des documents dans les principales langues du monde ainsi que dans des paires de langues moins courantes.
- **Traitement sécurisé** : Les fichiers sont transférés via des connexions HTTPS/SSL chiffrées, traités de manière sécurisée, puis automatiquement supprimés du serveur dans les 24 heures suivant la traduction.
- **Limite d'envoi de PDF de 20MB** : Prend en charge les fichiers PDF jusqu'à 20MB par envoi.
