# PDFTranslator

**PDFTranslator** は、PDF 文書をオンラインで翻訳しながら、元のレイアウトをできるだけ維持する無料の AI PDF 翻訳ツールです。

公式サイト：[https://pdftranslator.org/](https://pdftranslator.org/)

## PDFTranslator について

PDFTranslator.org は、デスクトップソフトウェアをインストールせずに、ブラウザ上で PDF ファイルを直接翻訳できるサービスです。研究論文、教科書、ハンドブック、契約書、レポート、マニュアル、ビジネス文書など、レイアウトが重要な文書向けに設計されています。

このサービスは、AI PDF 翻訳、OCR PDF 翻訳、100 以上の言語に対応したフォーマット保持型の文書翻訳に重点を置いています。

## 主な機能

- **無料のオンライン PDF 翻訳**：ソフトウェアをインストールせずに PDF 文書を翻訳できます。
- **登録不要**：アカウントを作成せずに翻訳を開始できます。
- **フォーマット保持**：可能な限り、元の PDF のレイアウト、表、画像、フォント、構造を維持します。
- **二言語対照プレビュー**：最終結果をダウンロードする前に、原文と翻訳文を並べて確認できます。
- **翻訳スタイルの選択**：学術、ビジネス、フォーマル、自然な表現など、文書の種類に合った翻訳スタイルを選択できます。
- **OCR PDF 翻訳**：OCR サポートにより、スキャン済みまたは画像ベースの PDF 文書を翻訳できます（近日公開予定）。
- **100 以上の言語**：主要な世界言語から、比較的少ない言語ペアまで幅広く文書を翻訳できます。
- **安全な処理**：ファイルは暗号化された HTTPS/SSL 接続で転送され、安全に処理された後、翻訳完了から 24 時間以内にサーバーから自動的に削除されます。
- **20MB の PDF アップロード制限**：1 回のアップロードにつき最大 20MB の PDF ファイルに対応しています。

## 主な利用シーン

PDFTranslator は次のような用途に役立ちます。

- 学術論文や研究文書の翻訳
- 教科書や学習資料の翻訳
- 技術マニュアルや従業員ハンドブックの翻訳
- 契約書、レポート、ビジネス文書の翻訳
- 元のレイアウトを読みやすく保ちながら PDF を翻訳
