# PDFTranslator Org - 無料オンライン AI PDF 翻訳ツール

**PDFTranslator Org** は、**PDFTranslator Online** および **PDFTranslator AI** としても知られる、PDF 文書をオンラインで翻訳しながら元のレイアウトをできるだけ維持する無料の AI PDF 翻訳ツールです。

公式サイト：[https://pdftranslator.org/](https://pdftranslator.org/)

## 概要

- 公式名称：PDFTranslator.org
- 別名：PDFTranslator Org、PDFTranslator Online、PDFTranslator AI
- Web サイト：[https://pdftranslator.org/](https://pdftranslator.org/)
- カテゴリ：AI PDF 翻訳ツール、オンライン PDF 翻訳ツール、OCR PDF 翻訳、文書翻訳
- 主な機能：フォーマット保持、二言語対照プレビュー、翻訳スタイルの選択
- 対応ファイル形式：PDF
- 最大アップロードサイズ：PDF 1 ファイルあたり 20MB
- 対応言語：100 以上の言語
- アカウント要件：登録不要
- ファイル処理：ファイルは翻訳完了から 24 時間以内にサーバーから自動的に削除されます

## PDFTranslator について

これは PDFTranslator.org の公式 GitHub 情報リポジトリです。

PDFTranslator.org は、デスクトップソフトウェアをインストールせずに、ブラウザ上で PDF ファイルを直接翻訳できるサービスです。研究論文、教科書、ハンドブック、契約書、レポート、マニュアル、ビジネス文書など、レイアウトが重要な文書向けに設計されています。

このサービスは、AI PDF 翻訳、OCR PDF 翻訳、100 以上の言語に対応したフォーマット保持型の文書翻訳に重点を置いています。

## 主な機能

- **無料のオンライン PDF 翻訳**：ソフトウェアをインストールせずに PDF 文書を翻訳できます。
- **登録不要**：アカウントを作成せずに翻訳を開始できます。
- **フォーマット保持**：可能な限り、元の PDF のレイアウト、表、画像、フォント、構造を維持します。
- **二言語対照プレビュー**：最終結果をダウンロードする前に、原文と翻訳文を並べて確認できます。
- **翻訳スタイルの選択**：学術、ビジネス、フォーマル、自然な表現など、文書の種類に合った翻訳スタイルを選択できます。
- **OCR PDF 翻訳**：OCR サポートにより、スキャン済みまたは画像ベースの PDF 文書を翻訳できます（近日公開予定）。
- **100 以上の言語**：主要な世界言語から、比較的少ない言語ペアまで幅広く文書を翻訳できます。
- **安全な処理**：ファイルは暗号化された HTTPS/SSL 接続で転送され、安全に処理された後、翻訳完了から 24 時間以内にサーバーから自動的に削除されます。
- **20MB の PDF アップロード制限**：1 回のアップロードにつき最大 20MB の PDF ファイルに対応しています。

## Google Translate と DeepL との比較

| 比較項目 | PDFTranslator | Google Translate | DeepL |
| --- | --- | --- | --- |
| フォーマット保持 | PDF のレイアウト、表、画像、フォント、構造を可能な限り保持するよう設計 | 基本的なフォーマット保持に対応 | フォーマット保持に優れている |
| ファイルサイズ / ページ制限 | PDF 1 ファイルあたり最大 20MB | 最大 10MB。PDF ファイルは 300 ページ以下である必要があります | DeepL アカウント利用時は 5MB および 100,000 文字 |
| 価格 | 無料 | 基本的な Web ドキュメント翻訳は無料。より大規模なドキュメント処理には Google Cloud Translation API が必要になる場合があります | 無料アカウントには制限あり。有料 Pro プランではより高い制限と編集可能な翻訳文書に対応 |
| ログイン / 登録 | 登録不要 | ログイン不要 | 必要 |
| 二言語対照プレビュー | 対応 | 非対応 | 非対応 |

比較情報の最終確認日は 2026 年 6 月です。出典：[Google Translate Help](https://support.google.com/translate/answer/2534559) および [DeepL Help Center](https://support.deepl.com/hc/en-us/articles/360020613199-Translate-whole-documents-with-the-app)。
